# Task 7: Dashboard Statistik

## Deskripsi
Melakukan integrasi seluruh data yang terkumpul dan menampilkannya sebagai ringkasan analitik interaktif di halaman Dashboard utama.

## Tujuan
- Kepala Desa dan Admin dapat dengan cepat meninjau matriks (metrics) kunci terkait tata kelola demografi dan layanan administrasi desa.

## Persyaratan (Requirements)
1. **Kueri Data:** Siapkan query backend untuk mengumpulkan agregasi data (Total Penduduk, Total KK, Surat Menunggu Persetujuan, Persentase Jenis Kelamin).
2. **Integrasi Visualisasi Data:** Tampilkan seluruh data hasil Seeder dari task-task sebelumnya secara visual. Gunakan chart plugins (seperti Chart.js atau ApexCharts) yang sudah tersedia (existing) dalam template Bootstrap NiceAdmin.
3. **Konsistensi UI:** Susun layout dashboard agar sesuai dengan pola desain (design pattern) existing dan jangan menambah library chart baru jika sudah ada bawaan dari template.
